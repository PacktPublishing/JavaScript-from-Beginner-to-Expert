javascript is very needed because for example if you have a clock you would need to every 1 second request to server a new value. With Javascript everything is running on the user side. Imagine a situation where the clock would need to reset each second and your webpage would just constantly reload to update the content. With Javascript you can update the content without reloading the webpage. What's more there are advanced libaries for Javascript like AJAX that allows you to connect to database and retrieve info without reloading the website. So Javascript is run on the user's computer browser.
